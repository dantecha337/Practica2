# Proyecto python

Este proyecto contiene la resolucion de la practica 2, incluyendo los notebooks de cada ejercicio para su ejecucion

## Requisitos

- Python 3.x
- Jupyter Notebook

## Instalacion
1. Descargar el repositorio del proyecto
2. (Opcional) Crear un entorno virtual con "python -m venv .venv" y activarlo con ".venv/Scripts/activate"
3. Instalar dependencias con "pip install -r requirements.txt"

## Ejecucion
1. Abrir Jupyter Notebook con "jupyter notebook" o "jupyter lab" en la consola
2. Navegar a la carpeta notebooks
3. Abrir el notebook correspondiente al ejercicio 10
4. Ejecutar las celdas para ver el resultado
